# 콘서트 티켓 예약 시스템

### Milestone

1. 분석 및 설계, 개발 환경 준비
   - 요구사항 분석
   - 시퀀스 다이어그램 작성
   - API 명세 작성
   - ERD 설계
2. 기능 구현, 테스트 작성
   - 기본 기능 작성
   - 주요 기능 작성
   - 단위 테스트 / 통합 테스트 작성
3. 심화 기능 구현, 리팩토링
   - 다수의 요청 동기화
   - 토큰 리프레시 설정
   - 배치 / 스케줄러 개선
   - NestJS 테크닉을 통한 코드 개선
4. 부하 테스트, 병목 개선
   - 부하 테스트 분석 보고서 작성
   - 병목 지점 파악 및 개선
5. E2E 테스트 작성, 배포 준비
   - API 단위의 E2E 테스트 작성
   - 배포 전략 도입
   - 최종 문서화
